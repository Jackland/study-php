# 路由地址-URL

## 使用

### php 中使用

```php
$url = url(); // 推荐

echo $url->link('account/account', 'a=b&c=f'); // 极不建议使用
echo $url->link('account/account', ['a' => 'b', 'c' => 'f']); // 不建议使用
echo $url->to(['account/account', 'a' => 'b', 'c' => 'f']); // 建议使用
echo url(['account/account', 'a' => 'b', 'c' => 'f']); // 建议
// 构建形式
echo $url
    ->withRoute('account/account')
    ->withCurrentQueries()
    ->withQueries(['a' => 'b', 'x' => 'y'])
    ->withoutQueries(['a', 'x'])
    ->build();
```

### twig 中使用

```twig
# 不带参数
{{ url('account/account') }}
带参数
{{ url('account/account', {'a': 'b'}) }}
```
