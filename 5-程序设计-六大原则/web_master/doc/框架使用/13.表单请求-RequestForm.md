# RequestForm 表单请求

用于对表单提交页面进行整合，处理单个业务相关逻辑

减少原 controller 或 model 的负担

## 使用

建立 RequestForm

```php
<?php

namespace Demo;

use Framework\Model\RequestForm\RequestForm;

class DemoForm extends RequestForm
{
    // 自动载入表单的数据，变量名和 form 表单的 input name 一致
    // 必须为 public!!
    public $name;
    public $user_name;
    public $password;

    /**
     * @inheritDoc
     */
    protected function getRules(): array
    {
        // 需要校验的字段
        // 所有需要自动载入的字段必须定义，否则会导致数据载入不了!!
        return [
            'name' => 'required',
            'user_name' => 'required|max:16',
            'password' => 'nullable|min:10',
        ];
    }

    /**
     * @inheritDoc
     */
    protected function getAttributeLabels(): array
    {
        // 修改错误提示时的字段名
        // 可选修改，默认为变量名
        return [
            'user_name' => '用户名',
        ];
    }

    /**
     * @inheritDoc
     */
    protected function getAutoLoadRequestData()
    {
        // 自动载入的数据
        // 可选，默认为 request->attributes->all()
        return $this->request->post();
    }
    
    // 实际要执行的业务，名字自取
    public function doSomething()
    {
        // 判断校验
        if (!$this->isValidated()) {
            // 校验不通过返回错误信息
            return [
                'error' => $this->getFirstError(),
            ];
        }
        // 实际业务处理

        return [
            'name' => $this->name,
            'user_name' => $this->user_name,
            'password' => $this->password,
        ];
    }
}
```

在控制器中使用

```php
public function testRequestForm(DemoForm $form)
{
    $data = $form->doSomething();

    return $this->json($data);
}
```
